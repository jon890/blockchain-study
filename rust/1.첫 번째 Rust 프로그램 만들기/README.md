### 학습 목표
1. 함수, 데이터 형식 및 변수를 비롯한 핵심 Rust 언어 개념을 살펴본다.
2. 텍스트, 숫자, 부울 값 및 복합 데이터에 대한 기본 Rust 형식을 이해한다.
3. 기본 Rust 프로그램을 만들고 컴파일하고 실행한다.
4. 프로그램의 결과를 출력하는 방법을 알아본다
